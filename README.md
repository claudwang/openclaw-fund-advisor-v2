# 智能基金投资顾问 - OpenClaw AI

基于AI技术的智能基金投资顾问平台，提供专业的市场分析和投资建议。

## 🚀 功能特色

- **智能市场分析** - 基于深度学习的市场趋势分析
- **AI基金推荐** - 个性化投资方案推荐
- **风险控制管理** - 实时风险评估和预警
- **智能推送提醒** - 每日定时推送最新动态
- **投资组合监控** - 全面监控投资表现
- **多平台支持** - Web端和移动端访问

## 📦 快速安装

### 一键安装
```bash
bash -c "$(curl -s https://raw.githubusercontent.com/claudwang/openclaw-fund-advisor/main/install.sh)"
```

### 手动安装
1. 下载安装脚本
2. 运行配置向导
3. 启动服务

## 🔧 配置说明

配置文件路径：`~/.openclaw/config.yaml`

主要配置项：
- 风险偏好设置
- 投资金额配置
- 推送时间设置
- 数据源配置

## 📊 技术架构

- **前端**: HTML5, CSS3, JavaScript
- **后端**: Python + AI算法
- **数据源**: 东方财富网等权威数据源
- **部署**: GitHub Pages + CI/CD

## 🎯 使用指南

### 启动服务
```bash
openclaw start
```

### 查看状态
```bash
openclaw status
```

### 更新配置
```bash
openclaw config
```

## ⚠️ 风险提示

基金投资有风险，过往业绩不代表未来表现。本系统提供的投资建议仅供参考，不构成投资建议。投资者应根据自身风险承受能力谨慎决策。
